## Permission Values

- Visibility: `visible`, `hidden`, `masked`, `partial`.
- Operation: `allow`, `deny`, `read-only`, `approval-required`.
- Data scope: `all`, `region`, `store`, `supplier`, `self`, `custom`, `none`.
